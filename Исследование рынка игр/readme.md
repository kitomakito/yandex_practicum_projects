## Анализ  закономерностей, определяющих успешность продаж интернет-магазина
[Notebook link](https://github.com/kitomakito/yandex_practicum_projects/tree/d682df03f80038e053c5f21d0efbdb46043b3e21/%D0%98%D1%81%D1%81%D0%BB%D0%B5%D0%B4%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5%20%D1%80%D1%8B%D0%BD%D0%BA%D0%B0%20%D0%B8%D0%B3%D1%80)
* **Цель:** Используя исторические данные о продажах компьютерных игр, выявить закономерности, определяющие успешность продаж.

* **Что сделано:** Проведена предобработка данных, исследовательский анализ данных. Выбран актуальный период для анализа. Определен жизненный цикл продаж игр в зависимости от платформы, для которой она сделана И составлен прогноз на продажи следующего года. Визуализированы параметры, влияющие на продажи, построены графики корреляции, heatmap, boxplot, определены топ-5 платформ, жанров для каждого региона, составлен портреты пользователей. Проверены 2 гипотезы с использованием критерия Стьюдента для независимых выборок.
  
* **Итог:** Выявлены параметры, определяющие успешность продаж игры в разных регионах мира и параметры, на которые можно не обращать внимания. На основании этого подготовлен отчет для планирования рекламных кампаний. Определены наиболее привлекательные платформы с окном актуальности игр в 1-3 года.
  
* **Стек: Python, Pandas, NumPy, Matplotlib, Seaborn, исследовательский анализ данных (EDA), описательная статистика, предобработка данных, проверка статистических гипотез**
