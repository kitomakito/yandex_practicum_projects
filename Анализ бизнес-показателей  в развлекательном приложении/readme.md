### Описание проекта
В нашем распоряжении данные для анализа развлекательного приложения Procrastinate Pro+. Несмотря на огромные вложения в рекламу, последние несколько месяцев компания терпит убытки.
Есть данные о пользователях, привлечённых с 1 мая по 27 октября 2019 года:
- лог сервера с данными об их посещениях,
- выгрузка их покупок за этот период,
- рекламные расходы.
### Цель проекта
Разобраться в причинах убытков и помочь компании выйти в плюс. Нам предстоит изучить:
- откуда приходят пользователи и какими устройствами они пользуются,
- сколько стоит привлечение пользователей из различных рекламных каналов;
- сколько денег приносит каждый клиент,
- когда расходы на привлечение клиента окупаются,
- какие факторы мешают привлечению клиентов
## Stack: 
Pandas | Matplotlib | Numpy | Seaborn | когортный анализ | продуктовые метрики | юнит-экономика
## Общий вывод
В рамках исследования определен портрет основного платящего пользователя (страна, устройство, канал привлечения). 
- Проведен анализ **конверсии в платёж**, определены наилучшие для привлечения платящих пользоваателей каналы (в разрезе рынков). 
- **Выявлена причина утечки костов** и где была самая высокая стоимость привлечения (САС): проблема в маркетинговой стратегии по Американскому рынку, где на протяжении нескольких месяцев после "точки невозврата" по **ROI** продолжались вливания в неэффективные направления.
- **Самые высокие косты** ушли на каналы которые показали в т.ч. и наихудшее удержание. Ставка маркетинга на привлечение пользователей iOS из США через TipTop не сыграла.
Рекомендовано:
* Остановить использование убыточных каналов (AdNonSense для европейского рынка и FaceBoom для американского) - они не окупаются
* Выбрать для привлечения пользователей альтернативные рекламные каналы: lambdaMediaAds для европейского рынка и RocketSuperAds для американского (TipTop был явно переоценён мркетологми)
* Осторожнее подходить к вопросу распределения рекламных бюджетов и диверсифицировать риски.
